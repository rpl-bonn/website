Vision-Language-Action (VLA) models like π₀ and OpenVLA promise general-purpose robot control from language instructions — but they need task-specific data to be useful. This thesis takes a hands-on, end-to-end approach: you will set up an **XLeRobot** — a dual-arm mobile manipulator [1] — build a corresponding simulation environment, design mobile manipulation tasks that specifically require spatial memory, collect demonstration data via teleoperation in both real and simulated settings, and fine-tune state-of-the-art VLA models on your collected dataset. The result: a complete pipeline from hardware to data to policy, and a benchmark comparing how well current foundation models handle memory-dependent mobile manipulation.
